# maze-dfs-threejs
3D maze game built with Three.js. The maze is procedurally generated using a **depth-first search (DFS)** algorithm. Your goal is to find the room marked in gold.

## Features

- Procedural maze generation (DFS)
- First-person movement and mouse look
- Toggleable flashlight
- Dynamic lighting and shadows

## Controls

| Key | Action |
|-----|--------|
| W / A / S / D | Move |
| Mouse | Look around |
| F | Toggle flashlight |
| Click | Start game |
| Esc | Halt game |

## How to run

1. Install dependencies:

```
npm install vite
```

2. Start the development server:

```
npx vite
```

3. Open in browser: `http://localhost:5173`

## Technologies

- [Three.js](https://threejs.org/) — 3D rendering
- [Vite](https://vitejs.dev/) — development server
- JavaScript
